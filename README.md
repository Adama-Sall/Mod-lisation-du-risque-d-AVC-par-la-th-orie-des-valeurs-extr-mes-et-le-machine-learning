# Mod-lisation-du-risque-d-AVC-par-la-th-orie-des-valeurs-extr-mes-et-le-machine-learning
Ce projet vise à modéliser et prédire le risque d’AVC en combinant la EVTet du ML. L’objectif est d’identifier les profils à risque extrême et d’améliorer la performance prédictive en couplant modélisation statistique e.  Ce travail a été réalisé dans le cadre d’un mémoire de Master en Mathématiques appliquées / Science des données.
## Objectifs

Analyser les facteurs de risque d’AVC

Modéliser les comportements extrêmes des variables de risque

Estimer des seuils de risque extrême

Construire des modèles prédictifs de survenue d’AVC

Comparer les approches EVT et machine learning
